# caiyan-cli

在终端里玩[猜盐](https://xiaoce.fun/)每日猜词挑战。

零依赖，装完即用。

## 安装

```bash
npm install -g caiyan-cli
```

## 使用

```bash
caiyan
```

## 操作

| 输入 | 作用 |
|------|------|
| 中文词语 + 回车 | 猜测 |
| `h` | 查看猜测记录（按相似度排序） |
| `q` | 退出 |

## 选项

```bash
caiyan --list   # 每次猜完自动展示排序列表
```

## 规则

- 每天一个隐藏词语，北京时间 0 点刷新
- 无限次猜测机会
- 每次返回相似度百分比，越高越接近答案
- 圆点颜色从灰→红→黄→绿，表示越来越近

## 卸载

```bash
npm uninstall -g caiyan-cli
```

## 数据来源

游戏数据来自 [猜盐](https://xiaoce.fun/)，本工具仅提供终端交互界面。
